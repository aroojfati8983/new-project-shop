# 🛍️ new-project-shop

A complete product management system with CRUD operations.

## Features
- View all products
- Add new products  
- Edit products
- Delete products

## Tech Stack
- HTML, CSS, JavaScript
- Node.js + Express
- MongoDB
- Axios

## Setup
1. Clone repository
2. Run `npm install`
3. Run `npm start`
4. Open `product.html`
